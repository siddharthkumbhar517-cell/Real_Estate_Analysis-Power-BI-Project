# 🏡 Real Estate Analytics Dashboard | Power BI

![Real Estate Analytics Dashboard](https://www.gssprojects.in/wp-content/uploads/2022/09/real-estate-company-in-Mysore.jpg)

## 📌 Project Overview
This Power BI project provides a comprehensive analysis of the real estate market, covering **property sales, pricing trends, agent performance, neighborhood insights, and commercial office analytics**.  
The dashboard enables data-driven decision-making for stakeholders such as real estate managers, investors, and analysts.

---

## 🎯 Key Objectives
- Analyze real estate sales and pricing trends
- Track agent performance and experience
- Evaluate neighborhood market conditions
- Understand owner–agent relationships
- Monitor commercial office market insights

---

## 📊 Dashboard Highlights

### 🔹 Real Estate Performance
- Total Listed Price
- Average Market Value
- Property Type & Listing Type distribution
- City-wise market comparison

### 🔹 Sales Trends
- Yearly, quarterly, monthly sales analysis
- Furnishing status impact on pricing
- Property size vs parking availability

### 🔹 Agent & Office Analytics
- Agent experience distribution
- Office-wise agent contribution
- Regional performance insights

### 🔹 Owner & Property Insights
- Properties per owner
- Owner-wise market value
- Owner–agent workload relationship

### 🔹 Neighborhood Analysis
- Crime rate & school rating impact
- Walkability score analysis
- Neighborhood-wise price trends

## 📂 Dataset Information
The dataset includes:
- Property details (type, price, size, location)
- Agent and office information
- Owner and neighborhood attributes
- Market indicators such as crime rate, income, and school ratings

*(Note: This dataset is used for analytical and learning purposes.)*
